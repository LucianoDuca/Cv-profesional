Instituto
San Vicente

GitHub Ver sitio
Landing Page
Secciones Académicas
Portal Deportivo — Club Sanvi

Desarrollo completo de una plataforma web institucional multipage para un colegio privado argentino. Incluye landing page, secciones académicas por nivel (Inicial, Primaria y Secundaria), área de inglés, galería de fotos, formulario de contacto y un portal deportivo (Club Sanvi) con múltiples disciplinas. Integra autenticación de usuarios, paneles de gestión diferenciados por rol, soporte multilenguaje (español/inglés) y optimización de assets. Desplegado en Netlify con servidor propio en Node.js para endpoints sensibles.

HTML5 — estructura semántica y accesibilidad (ARIA) CSS3 — diseño responsive mobile-first, animaciones con scroll, variables CSS JavaScript (ES6+) — Intersection Observer API, galería lightbox con navegación por teclado Node.js + Express.js — servidor backend para manejo de rutas y endpoints de correo Supabase — base de datos PostgreSQL, autenticación y almacenamiento en la nube Resend API — envío transaccional de emails Netlify — CI/CD, cabeceras HTTP personalizadas y despliegue continuo

GoRugby

GitHub Ver sitio
Partidos en vivo
Rankings & Torneos
Sistema de Predicciones

Aplicación web full-stack orientada a la comunidad rugbística, con seguimiento de partidos en vivo, predicciones, rankings de torneos globales y gestión de clubes. Integra múltiples APIs deportivas externas con un proxy seguro del lado del servidor para proteger credenciales. Soporta temas claro/oscuro, diseño responsive y sistema de roles (usuario/admin).

React 18 + TypeScript — SPA con tipado estricto, componentización modular y Context API para estado global Vite — Build tool con hot reload, configuración de proxies y optimización de bundles CSS personalizado (Design System) — temas dinámicos claro/oscuro, 6 paletas de acento configurables Node.js + Vercel Edge Functions — proxy de APIs externas con caching HTTP (stale-while-revalidate) Integración de 4 APIs deportivas — api-sports.io, ESPN, Highlightly, TheSportsDB Vercel — Deploy continuo con SPA routing y funciones edge en producción

Consulta de Precios
Escaneo de Código de Barras
Gestión de Catálogo

Aplicación de escritorio desarrollada para una fiambrería local, que permite consultar precios en tiempo real mediante escaneo de código de barras. El negocio puede gestionar su catálogo completo de productos desde la misma herramienta, con sincronización automática a la nube.

React + TypeScript — componentes funcionales, hooks y tipado estático Electron — empaqueta la app web como aplicación nativa de Windows con instalador .exe Supabase (PostgreSQL) — base de datos en la nube con CRUD completo de productos Vite — bundler moderno con integración directa al proceso de Electron Web Audio API — feedback sonoro nativo al escanear un producto sin dependencias externas CSS personalizado — soporte de modo oscuro/claro con clases condicionales en React
