01

Instituto
San Vicente

GitHub Ver sitio

Plataforma web institucional multipage para un colegio privado argentino. Cubre desde la presentación pública hasta la gestión interna diferenciada por rol, incluyendo un portal deportivo completo y soporte bilingüe.

Funcionalidades

  • Autenticación con 3 niveles de rol — Administrador, Directivo y Docente
  • Panel de gestión de contenido diferenciado por tipo de usuario
  • Secciones académicas por nivel: Inicial, Primaria y Secundaria
  • Portal deportivo Club Sanvi con múltiples disciplinas
  • Galería de fotos con lightbox y navegación por teclado
  • Formulario de contacto con envío real de emails (Resend API)
  • Soporte bilingüe Español / Inglés integrado sin librerías externas
  • Diseño responsive optimizado para mobile, tablet y desktop
HTML5 · CSS3 · JavaScript (ES6+) Node.js + Express.js — backend para rutas y endpoints de correo Supabase — PostgreSQL, autenticación y almacenamiento en la nube Resend API · Netlify CI/CD
02

GoRugby

GitHub Ver sitio

Próximamente

Interfaz visual en desarrollo

Plataforma web full-stack orientada a la comunidad rugbística global. Consolida datos de múltiples fuentes externas en un solo lugar, protegiendo las API keys con un proxy seguro del servidor.

Funcionalidades

  • Seguimiento de partidos en vivo con datos actualizados en tiempo real
  • Sistema de predicciones de resultados con ranking de aciertos
  • Rankings de torneos globales: World Rugby, Premiership, Top 14, URC y más
  • Gestión de clubes y consulta de estadísticas por equipo
  • Proxy seguro del lado del servidor para proteger 4 API keys deportivas
  • Design system propio: temas claro/oscuro y 6 paletas de color configurables
  • Sistema de roles: usuario estándar y administrador
React 18 + TypeScript — SPA con Context API y tipado estricto Vite · CSS Design System personalizado Node.js + Vercel Edge Functions — proxy de APIs con caching HTTP 4 APIs deportivas integradas · Vercel CI/CD
03

Fiambrería
Duca

GitHub Instalador

Aplicación de escritorio nativa para Windows, desarrollada para una fiambrería local. Reemplaza el método manual de consulta de precios con un sistema rápido de escaneo y gestión de catálogo sincronizado en la nube.

Funcionalidades

  • Consulta de precios en tiempo real mediante escaneo de código de barras
  • Gestión completa del catálogo: agregar, editar, eliminar y buscar productos
  • Calculadora integrada para operaciones en el punto de venta
  • Sincronización automática con Supabase PostgreSQL en la nube
  • Feedback sonoro al escanear: tono de éxito o error (Web Audio API)
  • Instalador .exe nativo para Windows generado con electron-builder
  • Modo oscuro y claro con persistencia de preferencia del usuario
React + TypeScript · Electron — app nativa Windows Supabase (PostgreSQL) — CRUD en la nube Vite · Web Audio API · CSS modo oscuro/claro
